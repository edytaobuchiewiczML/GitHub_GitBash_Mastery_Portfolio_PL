# GitHub_GitBash_Mastery_Portfolio_PL
Repozytorium dokumentujące naukę Gita i GitHuba. Obejmuje: konfigurację, repozytoria, przestrzenie Git (Working Tree, Staging Area), rewizje (commit, log, diff), gałęzie (branch, merge, rebase), rozwiązywanie konfliktów, .gitignore, reset, revert, zdalne repozytoria (clone, push, fetch, pull) oraz pracę z GitHubem.
